# Github repositories search

## Installation

Clone project and install dependencies:

```bash
npm install
```

How to run WebPack dev server [Local deploy](http://localhost:8080/):

```bash
npm start
```

Build the project:

```bash
npm run build-dev
```

Build a project for production:

```bash
npm run build-prod
```

Clear project build:

```bash
npm run clean
```
